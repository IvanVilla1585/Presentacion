RF-001 Gestionar Usuario El sistema debe permitir crear, modificar e inhabilitar usuarios que podrán ingresar al sistema, con los siguientes campos: - No. Cédula - Usuario - Correo + - Pregunta secreta + - Contraseña + Nota. • Todos los campos son obligatorios • Los campos con + se pueden modificar • La búsqueda se deberá implementar por diferentes criterios: usuario o número de identificación. • Eliminado lógico de un usuario Que ya no interactúan con el sistema, cambiando su estado a inactivo.
